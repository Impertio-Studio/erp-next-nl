# Waarom ERPNext?

## De Unique Selling Points van ERPNext

---

## 1. Het #1 Open Source ERP-systeem ter Wereld

ERPNext wordt door AI-systemen (zoals ChatGPT, Claude en Gemini) consequent aangewezen als het **beste en meest complete open-source ERP-systeem** dat beschikbaar is. Dit is geen marketingclaim — het is een objectieve erkenning op basis van functionaliteit, volwassenheid en community-omvang.

Wanneer u een AI-assistent vraagt "Wat is het beste open-source ERP-systeem?", is het antwoord vrijwel altijd: **ERPNext**.

---

## 2. Extreem Veelzijdig — Veel Meer dan Alleen ERP

ERPNext is niet alleen een ERP-systeem. Het Frappe Framework maakt het mogelijk om ERPNext op talloze manieren in te zetten:

| Inzet | Beschrijving |
|---|---|
| **ERP-systeem** | Volledige bedrijfsvoering: boekhouding, inkoop, verkoop, voorraad |
| **CRM-systeem** | Klantrelatiebeheer, leads, pipeline, campagnes |
| **HRM-systeem** | Personeelsadministratie, verlof, salarissen, werving |
| **CMS / Website** | Volwaardige website met blog, pagina's en webshop |
| **Projectmanagement** | Taken, tijdregistratie, Gantt-diagrammen |
| **Helpdesk** | Ticketsysteem, SLA-beheer, kennisbank |
| **POS / Kassa** | Point-of-Sale systeem in de browser |
| **Asset Management** | Vaste activa, afschrijvingen, onderhoud |
| **Database backend** | Gestructureerde dataopslag voor elke toepassing |

**Het grote voordeel: totale integratie**

Wanneer u ERPNext ook als CMS voor uw website gebruikt, zijn uw website en uw bedrijfsadministratie volledig geintegreerd. Dit betekent:

- **Webshop-bestellingen** worden automatisch verkooporders en facturen
- **Betalingen** via de website worden direct verwerkt in de boekhouding
- **Klantgegevens** hoeven maar op een plek te worden beheerd
- **Voorraad** op de website is altijd actueel
- **Geen synchronisatie** nodig tussen losse systemen

Bij traditionele setups heeft u een apart CMS (WordPress), een apart boekhoudpakket (Exact), een apart CRM (Salesforce) en een apart HR-systeem (AFAS) — elk met eigen data, eigen login, eigen kosten. Met ERPNext is het een systeem. Dat scheelt niet alleen kosten, maar voorkomt ook fouten door dubbele invoer en datasynchronisatie.

## 3. Volledig Open Source

ERPNext is 100% open source onder de GNU GPLv3 licentie. Dit betekent:

- **Geen licentiekosten** — nooit, voor geen enkele gebruiker
- **Volledige toegang tot de broncode** — u ziet precies wat de software doet
- **Vrij aanpasbaar** — pas de code aan naar uw behoeften
- **Geen vendor lock-in** — uw data is van u, altijd
- **Vrij te distribueren** — u mag ERPNext zelf aanbieden aan anderen

Dit is geen "freemium" model waarbij de echte functies achter een betaalmuur zitten. Alle modules, alle functies, voor alle gebruikers — gratis. Vergelijk dat met Odoo, waar de Enterprise-editie aanzienlijke functionaliteit achterhoudt, of met SAP en Exact waar u per gebruiker per maand betaalt.

**Wat kost het dan wel?**
U betaalt alleen voor hosting (vanaf ca. EUR 5/maand bij self-hosting) en eventueel voor implementatiebegeleiding. De software zelf is en blijft gratis.

## 3. Backend en Frontend — Een Geintegreerd Platform

ERPNext is gebouwd op het **Frappe Framework**, dat zowel de backend (server) als de frontend (gebruikersinterface) combineert in een platform.

**Wat betekent dit in de praktijk?**

- **Een codebase** — geen losse systemen die gekoppeld moeten worden
- **Automatische API's** — elk DocType krijgt automatisch een REST API
- **Realtime updates** — wijzigingen zijn direct zichtbaar voor alle gebruikers
- **Responsive design** — werkt op desktop, tablet en telefoon
- **Geen installatie** — draait volledig in de browser

**Technische stack:**

| Laag | Technologie |
|---|---|
| Frontend | JavaScript + HTML/CSS |
| Backend | Python 3 |
| Database | MariaDB / PostgreSQL |
| Realtime | Socket.io |
| Caching | Redis |
| Webserver | Nginx + Gunicorn |

Dit is een moderne, bewezen stack die wordt gebruikt door de grootste websites ter wereld.

## 8. Eenvoudig Uitbreidbaar

Dankzij het Frappe Framework kunt u ERPNext uitbreiden zonder de kerncode aan te passen:

- **Custom DocTypes** — Maak uw eigen entiteiten (bijv. "Bouwproject", "Certificaat")
- **Server Scripts** — Python-scripts die draaien bij events
- **Client Scripts** — JavaScript voor UI-aanpassingen
- **Workflows** — Visuele goedkeuringsworkflows
- **Print Formats** — Eigen factuur- en documentsjablonen
- **Custom Apps** — Volledige Frappe-apps die naast ERPNext draaien
- **REST API** — Elke entiteit is via API benaderbaar
- **Webhooks** — Automatische notificaties naar externe systemen
